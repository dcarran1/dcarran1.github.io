The Market Landscape

In 2025, the main browsers dominating the internet include Google Chrome, Safari, Microsoft Edge, and Firefox. Google chrome currently holds about 67% of the global market share which means that about two out of three users are likely using Google Chrome as their main browser (Pawel, 2025).

Furthermore, since Google Chrome is the current dominant leader in internet browsing, developers will usually test websites and applications on Google Chrome first and then test on other browsers. Therefore, many websites today usually work best on Google Chrome (Cano, 2025). When Google Chrome is not being used, there is always Safari, with a global market share of about 19% and it tends to be used quite often on mobile devices. Safari does have a good advantage in that Apple requites all IOS browsers to use WebKit which is Safari’s browser engine. Since it uses WebKit, Chrome and Firefox on all iPhones have to use the same engine.

Microsoft edge typically uses chromium though it used to have it’s own engine prior to 2020 (Cano, 2025). This was a shift in order to remain compatible with most other websites as the browser landscape continued to develop, Microsoft Edge focused on adhering to the browser standards of the time instead of having to play catch up. Lastly, there if Firefox which has been widely known for its privacy protections and customization features, however, Firefox has remained with the fewest extensions out of all of the browsers.
